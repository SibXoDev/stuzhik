# Управление модами

## Установка модов

### Из каталога

1. Открой экземпляр
2. Вкладка "Моды" → "Каталог"
3. Найди мод через поиск
4. Кнопка "Установить"

Stuzhik автоматически:
- Выберет версию для твоей версии Minecraft
- Проверит совместимость с загрузчиком
- Предупредит о конфликтах
- Установит зависимости

### Из файла

Перетащи `.jar` файл в окно или:

1. Кнопка "Добавить локальный мод"
2. Выбери `.jar` файл
3. Готово

### Зависимости

Если мод требует другие моды:

1. Stuzhik покажет список зависимостей
2. Кнопка "Установить все"
3. Все зависимости установятся автоматически

## Управление модами

### Включение/отключение

1. Список модов → клик на переключатель
2. Мод переименуется в `.jar.disabled`
3. При следующем запуске не загрузится

### Обновление

**Вручную:**
1. Клик на мод → "Обновить"
2. Stuzhik найдёт последнюю версию
3. Заменит файл

**Автоматически:**
1. Клик на мод → "Настройки"
2. Включи "Автообновление"
3. При запуске проверяется новая версия

### Удаление

1. Клик на мод → "Удалить"
2. Или выбери несколько → "Удалить выбранные"

## Поиск модов

### Фильтры

- **Категория:** Tech, Magic, Decoration, и т.д.
- **Загрузчик:** Forge/Fabric/NeoForge/Quilt
- **Версия MC:** Только совместимые
- **Источник:** Modrinth/CurseForge

### Сортировка

- По популярности (downloads)
- По дате обновления
- По рейтингу

## Конфликты модов

### Автоматическая проверка

При установке Stuzhik проверяет:

| Тип конфликта | Пример |
|---------------|--------|
| Несовместимость | OptiFine + Sodium |
| Дубликаты | Sodium + Rubidium |
| Loader mismatch | Forge мод на Fabric |
| Версия MC | Мод для 1.19 на 1.20 |

### Решение конфликтов

Если конфликт найден:

1. **Critical** - установка заблокирована
2. **Warning** - можно установить, но будет краш
3. **Info** - работать будет, но могут быть баги

Stuzhik предложит:
- Удалить конфликтующий мод
- Установить альтернативу
- Игнорировать (на свой риск)

```

macOS:   ~/Library/Application Support/minecraft-launcher/instances/<id>/mods/
```

## Troubleshooting

### Мод не появляется в игре

**Проверь:**
1. Правильный загрузчик? (Forge мод на Forge)
2. Правильная версия MC?
3. Мод включён? (не `.disabled`)
4. Есть зависимости? (Fabric API для Fabric)

### Краш после установки мода

1. Log Analyzer автоматически найдёт виновный мод
2. Удали мод через Stuzhik
3. Или откати версию: клик на мод → "Откатить"

### Мод не обновляется

1. Проверь что автообновление включено
2. Может быть новая версия только для новой MC
3. Dev Console (`Ctrl+Shift+D`) → есть ошибки?
