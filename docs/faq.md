# FAQ

## Общие вопросы

### Чем отличается от других лаунчеров?

**Анализ крашей:** Находит виноватые моды и предлагает решения. Другие лаунчеры только показывают лог.

**Производительность:** Быстрый запуск, малое потребление ресурсов.

**Live мониторинг:** Отслеживает проблемы во время игры.

### Поддерживается ли Fabric/Forge/NeoForge/Quilt?

Да, все 4 загрузчика + Vanilla.

### Можно ли использовать с существующими экземплярами?

Да. Импорт из:
- MultiMC/Prism Launcher
- Modrinth модпаки (.mrpack)
- CurseForge модпаки (.zip)

### Требуется ли аккаунт Minecraft?

Да, нужно использовать Microsoft аккаунт с купленным Minecraft: Java Edition.

## Производительность

### Почему Forge устанавливается долго?

Forge installer загружает библиотеки с `maven.minecraftforge.net`. Это занимает 1-3 минуты.

Stuzhik использует зеркала BMCLAPI для ускорения.

### Сколько места занимает экземпляр?

| Компонент | Размер |
|-----------|--------|
| Minecraft | ~200MB |
| Forge/Fabric | ~50MB |
| Моды (10-50шт) | ~100-500MB |
| **Итого** | ~400-800MB |

### Можно ли запустить несколько экземпляров одновременно?

Да, без ограничений.

## Моды

### Как установить мод?

**Вариант 1:** Каталог
1. Открой экземпляр → Моды → Каталог
2. Найди мод → Установить

**Вариант 2:** Файл
1. Открой экземпляр → Моды
2. Кнопка "Добавить" или drag&drop `.jar`

### Stuzhik автоматически обновляет моды?

По умолчанию нет. Можно включить для каждого мода:
1. Список модов → клик на мод
2. Включи "Автообновление"

### Что делать при конфликте модов?

Stuzhik предупредит ДО установки если знает о конфликте.

Если краш уже произошёл:
1. Анализатор логов → найдёт виновный мод
2. Кнопка "Удалить мод" или "Заменить на совместимый"

### Где хранятся моды?

На Windows:
```
C:\Users\<username>\AppData\Roaming\ru.sibxodev.stuzhik/instances/<instance_id>/mods/
```

## Проблемы

### OutOfMemoryError

**Причина:** Недостаточно RAM выделено.

**Решение:**
1. Открой экземпляр → Edit
2. Увеличь RAM (рекомендуется 4-6GB для модпаков)
3. Сохрани → Перезапусти

### Игра не запускается

**Проверь:**
1. Dev Console → есть ли ошибки?
2. Log Analyzer → найдёт проблему автоматически
3. Java установлена? (Stuzhik устанавливает автоматически, но может быть ошибка)

### Моды не загружаются

**Для Forge:**
- Файл должен быть для Forge (в названии `-forge-`)
- Версия Minecraft должна совпадать

**Для Fabric:**
- Требуется Fabric API (устанавливается автоматически)
- Файл для Fabric (в названии `-fabric-`)

### Краш с "Mixin"

**Причина:** Конфликт модов, модифицирующих одинаковый код.

**Решение:**
1. Log Analyzer покажет конфликтующие моды
2. Удали один из конфликтующих
3. Или используй альтернативный мод
