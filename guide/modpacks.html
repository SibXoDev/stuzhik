Модпаки ​

Формат Stuzhik (.stzhk) ​

Stuzhik использует собственный формат модпаков .stzhk - оптимизированный для быстрой установки и максимальной совместимости.

Преимущества ​

  • Быстрая установка - оптимизированная структура
  • Полная совместимость - все настройки сохраняются
  • Компактный размер - эффективное сжатие
  • Метаданные - версия, автор, описание, скриншоты

Экспорт в .stzhk ​

  1. Открой экземпляр
  2. Кнопка "Экспорт" → "Stuzhik (.stzhk)"
  3. Заполни метаданные
  4. Сохрани файл

Импорт .stzhk ​

Просто перетащи .stzhk файл в окно Stuzhik или:

  1. Главная → "Импорт"
  2. Выбери файл
  3. Готово

Совместимость с другими форматами ​

Stuzhik также поддерживает импорт из популярных форматов:

ФорматИсточникПоддержка
.stzhkStuzhikПолная
.mrpackModrinthПолная
.zipCurseForgeПолная

Импорт из Modrinth ​

  1. Главная → "Модпак"
  2. Поиск по названию
  3. Выбери версию → "Установить"

Импорт из CurseForge ​

  1. Скачай .zip с CurseForge
  2. Stuzhik → "Импорт"
  3. Выбери файл

Создание модпака ​

Что включается ​

  • Список модов с версиями
  • Конфигурации модов
  • Настройки игры
  • Ресурспаки и шейдеры (опционально)

Что НЕ включается ​

  • Minecraft (скачивается отдельно)
  • Загрузчик (устанавливается автоматически)
  • Сохранения миров

Редактор модпаков ​

Сравнение версий ​

  1. Модпак → "Редактор"
  2. Загрузи две версии
  3. Увидишь:
    • Добавленные моды
    • Удалённые моды
    • Обновлённые моды

Обновление ​

  1. Открой экземпляр с модпаком
  2. "Проверить обновления"
  3. Применить изменения

Серверные модпаки ​

При экспорте для сервера Stuzhik автоматически:

  • Исключает client-only моды
  • Добавляет server-only моды
  • Настраивает server.properties

Синхронизация с друзьями ​

  1. Экспортируй модпак в .stzhk
  2. Отправь друзьям
  3. Они импортируют в Stuzhik
  4. Все на одинаковых версиях

Решение проблем ​

Установка зависла ​

Dev Console (Ctrl+Shift+D) покажет детали. Возможные причины:

  • Медленное соединение
  • Недоступный мод

Краш после установки ​

Log Analyzer автоматически найдёт проблемные моды и предложит решения.
