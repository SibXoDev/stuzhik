# Запусти установщик
# Готово

bash
# AppImage
chmod +x stuzhik_linux_x64.AppImage
./stuzhik_linux_x64.AppImage

# Или .deb
sudo dpkg -i stuzhik_linux_amd64.deb

bash
# Скачай .dmg
# Перетащи в Applications
# При первом запуске: System Settings → Privacy & Security → Open Anyway

Первый запуск ​

  1. Создай экземпляр

    • Кнопка "Создать" в header
    • Выбери версию Minecraft (например, 1.20.1)
    • Выбери загрузчик (Forge/Fabric/NeoForge/Quilt)
    • Нажми "Создать"

  2. Установка займёт 1-3 минуты

    • Java скачается автоматически
    • Minecraft установится из официальных источников
    • Загрузчик установится (Forge может занять ~2 мин)

  3. Запусти игру

    • Нажми "Играть" на карточке экземпляра
    • Первый запуск ~30 секунд (генерация ресурсов)

Добавление модов ​

Из каталога ​

  1. Открой экземпляр
  2. Вкладка "Моды"
  3. Кнопка "Каталог"
  4. Найди нужный мод
  5. Нажми "Установить"

Stuzhik автоматически:

  • Проверит совместимость с версией MC
  • Предупредит о конфликтах с установленными модами
  • Установит недостающие зависимости

Из файла ​

  1. Перетащи .jar файл в папку модов
  2. Или используй кнопку "Добавить локальный мод"

Анализатор логов ​

При краше:

  1. Лаунчер автоматически откроет анализатор
  2. Увидишь список найденных проблем
  3. Для каждой проблемы - возможные решения
  4. Кнопка "Применить" для автоисправления (где возможно)
